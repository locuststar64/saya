Probe Generator

Quick Start:

  • Enter a URL for an image on the Internet in the Image URL field.
  • Wait a few seconds for the Probe URL to be generated.
  • Copy the Probe URL and use it to post the image.

Guidelines:

  • Post images in the context of the discussion.
  • Use one probe per post or signature. People do not normally post the same image twice.
  • Use smaller pictures. Some forums download images locally and resize them if they are too big. That would remove both Saya and the original site form the loop.
  • Use a temporary account when possible. Even though this service is very hard to detect, it is always best to be careful not to blow your cover.

Best Practices:

  • Post on the first page of a thread.
  • Use a signature. It is not searchable.
